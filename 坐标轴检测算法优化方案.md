# 坐标轴检测算法优化方案

## 1. 问题分析

### 1.1 原有算法的局限性
- **鲁棒性差**：依赖形态学长线搜索，对不同分辨率/样式的图像适应性差
- **刻度线定位不准确**：无法精确定位刻度线位置（200, 300, 400...），只能靠"比例推算"
- **映射偏移问题**：不同裁剪方式和边距下会产生映射偏移
- **硬编码问题**：X轴范围固定为200-700nm、Y轴固定为0-4.5，不具备适应性

## 2. 优化方案

### 2.1 核心改进点

#### 2.1.1 更鲁棒的坐标轴检测
- **边缘检测**：使用Sobel边缘检测算法，增强对不同对比度图像的适应能力
- **直线检测**：基于霍夫变换原理的简化实现，能够检测不同粗细的坐标轴
- **长度筛选**：通过长度筛选确保找到的是真正的坐标轴，而非其他短线段
- **容错机制**：允许一定程度的角度偏差，提高对倾斜图像的适应能力

#### 2.1.2 精确定位刻度线
- **刻度线检测**：专门的刻度线检测算法，识别坐标轴上的短线段
- **长度阈值**：通过长度阈值区分刻度线和其他噪声
- **方向判断**：根据坐标轴方向判断刻度线的方向（x轴刻度为垂直线，y轴刻度为水平线）

#### 2.1.3 解决映射偏移问题
- **原点计算**：精确计算坐标轴交点作为原点
- **相对映射**：基于原点和坐标轴长度进行相对映射，而非绝对位置
- **自适应范围**：虽然默认使用200-700nm/0-4.5范围，但架构支持未来扩展OCR识别实际刻度值

### 2.2 算法流程

1. **图像预处理**：读取图像并转换为灰度图
2. **边缘检测**：使用Sobel算子检测图像边缘
3. **直线检测**：检测图像中的水平和垂直直线
4. **坐标轴识别**：找到最长的水平直线作为x轴，最长的垂直直线作为y轴
5. **刻度线检测**：检测坐标轴上的刻度线
6. **原点计算**：计算坐标轴交点作为原点
7. **曲线提取**：在坐标轴范围内提取光谱曲线
8. **物理坐标转换**：将像素坐标转换为真实的波长和吸光度值

img.Image detectEdges(img.Image grayImage) {
  // 使用Sobel边缘检测
  final edges = img.sobel(grayImage);
  // 二值化边缘图像
  return img.threshold(edges, 128);
}

  final lines = <Line>[];
  final width = edgeImage.width;
  final height = edgeImage.height;
  
  // 检测水平直线
  for (int y = 0; y < height; y++) {
    int lineLength = 0;
    int startX = 0;
    
    for (int x = 0; x < width; x++) {
      final pixel = img.getPixel(edgeImage, x, y);
      if (pixel > threshold) {
        if (lineLength == 0) {
          startX = x;
        }
        lineLength++;
      } else {

#### 物理坐标转换
```dart
List<Map<String, double>> convertToPhysicalCoordinates(
  List<Point> curvePixels,
  Point origin,
  Line xAxis,
  Line yAxis,
  double xMin,
  double xMax,
  double yMin,
  double yMax,
) {
  final spectralData = <Map<String, double>>[];
  
  // 计算映射关系
  final xRange = xMax - xMin;
  final yRange = yMax - yMin;
  
  final xPixelRange = xAxis.length();
  final yPixelRange = yAxis.length();
  
  final xScale = xRange / xPixelRange;
  final yScale = yRange / yPixelRange;
  
  // 转换每个像素点
  for (final point in curvePixels) {
    // 计算相对于原点的偏移
    final xOffset = point.x - origin.x;
    final yOffset = point.y - origin.y;
    
    // 计算物理坐标
    final wavelength = xMin + xOffset * xScale;
    // 注意：图像y轴向下，所以吸光度需要反转
    final absorbance = yMax - yOffset * yScale;
    
    // 过滤超出范围的数据
    if (wavelength >= xMin && wavelength <= xMax &&
        absorbance >= yMin && absorbance <= yMax) {
      spectralData.add({
        'wavelength': wavelength,
        'absorbance': absorbance,
      });
    }
  }
  
  return spectralData;
}
```
