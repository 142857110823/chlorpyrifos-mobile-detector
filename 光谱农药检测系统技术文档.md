# 光谱农药检测系统技术文档

## 1. 系统概述

### 1.1 系统简介
光谱农药检测系统是一个基于Flutter开发的移动应用，能够通过分析PNG/JPG格式的光谱曲线图图片，识别四种不同农药（吡虫啉、扑虱灵、种衣剂、苄·二氣）的残留情况。系统采用MobileNetV2模型进行图像分类，实现离线、快速、准确的农药残留检测。

### 1.2 技术架构
- **前端**：Flutter 3.0+
- **模型**：MobileNetV2（1.0×宽度乘数）
- **模型部署**：TFLite（INT8量化）
- **图像处理**：Dart image库
- **权限管理**：permission_handler
- **图片选择**：image_picker

## 2. 系统功能

### 2.1 核心功能
- **光谱图片选择**：支持相机拍摄和本地相册选择
- **图片预处理**：包括灰度化、去噪、二值化和缩放
- **模型推理**：使用MobileNetV2模型进行多分类
- **结果展示**：显示检测结果和置信度
- **错误处理**：基本的错误处理和提示

### 2.2 辅助功能
- **权限管理**：适配不同Android版本的权限要求
- **离线运行**：所有处理在本地完成，无需网络连接
- **用户界面**：美观、简洁，操作流程简单直观

#### 3.1.2 关键代码

**光谱图片检测页面**：
```dart
// 核心检测逻辑
Future<void> _detectPesticide() async {
  if (_selectedImage == null) return;

  try {
    // 1. 读取图片
    img.Image? originalImage = img.decodeImage(_selectedImage!.readAsBytesSync());
    if (originalImage == null) {
      throw Exception("无法解码图片");
    }
    
    // 2. 图片预处理
    img.Image processedImage = _preprocessImage(originalImage);
    
    // 3. 转换为模型输入格式
    var input = List.filled(1 * 224 * 224 * 3, 0.0).reshape([1, 224, 224, 3]);
    for (int y = 0; y < 224; y++) {
      for (int x = 0; x < 224; x++) {
        var pixel = processedImage.getPixel(x, y);
        // 对于二值化图像，RGB值相同，使用灰度值
        double grayValue = img.getRed(pixel) / 255.0;
        input[0][y][x][0] = grayValue;
        input[0][y][x][1] = grayValue;
        input[0][y][x][2] = grayValue;
      }
    }

    // 4. 模型推理
    var output = List.filled(4, 0.0).reshape([1, 4]);
    _interpreter.run(input, output);
    
    // 5. 解析结果
    List<double> probabilities = output[0];
    int predictedClass = probabilities.indexOf(probabilities.reduce((a, b) => a > b ? a : b));
    double confidence = probabilities[predictedClass] * 100;
    
    // 6. 获取农药名称
    String pesticideName = _labelMap.containsKey(predictedClass.toString()) 
        ? _labelMap[predictedClass.toString()]! 
        : "未知农药";

    // 7. 显示结果
    setState(() {
      _result = "✅ 检测结果：存在**$pesticideName**残留\n置信度：${confidence.toStringAsFixed(1)}%";
      _isLoading = false;
    });
  } catch (e) {
    print("检测失败: $e");
    setState(() {
      _result = "检测失败: ${e.toString()}";
      _isLoading = false;
    });
  }
}
```

### 3.2 模型实现

#### 3.2.1 模型架构
- **基础模型**：MobileNetV2（1.0×宽度乘数）
- **分类头**：GlobalAveragePooling2D + Dense(128) + Dense(4, softmax)
- **输入尺寸**：224×224×3
- **输出**：4个类别（吡虫啉、扑虱灵、种衣剂、苄·二氣）

#### 3.2.2 模型训练
```python
def create_model(num_classes):
    """
    创建多分类模型
    使用MobileNetV2作为基础模型
    基于MobileNetV2核心原理：深度可分离卷积、倒置残差块、线性瓶颈
    """
    # 加载预训练的MobileNetV2模型
    base_model = MobileNetV2(
        input_shape=(*IMG_SIZE, 3),
        include_top=False,
        weights="imagenet",
        alpha=1.0  # 宽度乘数，控制通道数
    )
    
    # 冻结预训练权重
    base_model.trainable = False
    
    # 添加分类头
    x = base_model.output
    x = GlobalAveragePooling2D()(x)
    # 保持线性瓶颈设计理念，避免在低维空间使用ReLU
    x = Dense(128)(x)  # 线性激活，无ReLU
    output = Dense(num_classes, activation="softmax")(x)
    
    model = Model(inputs=base_model.input, outputs=output)
    
    # 编译模型
    model.compile(
        optimizer=Adam(learning_rate=LEARNING_RATE),
        loss="categorical_crossentropy",
        metrics=["accuracy"]
    )
    
    return model
```

### 4.3 问题与解决方案

#### 4.3.1 中文路径编码问题
- **问题**：OpenCV无法处理中文路径
- **解决方案**：使用PIL库读取和保存图片，支持中文路径

#### 4.3.2 数据集过小问题
- **问题**：每个类别只有1个原始样本
- **解决方案**：通过数据增强扩充数据集，包括旋转、亮度调整、添加噪声等

#### 4.3.3 模型训练准确率问题
- **问题**：模型准确率不足
- **解决方案**：使用MobileNetV2预训练模型，冻结基础权重，只训练分类头

#### 4.3.4 语法错误问题
- **问题**：代码中存在颜色方法调用错误
- **解决方案**：将`withValues`改为正确的`withOpacity`方法

#### 4.3.5 权限请求适配问题
- **问题**：Android 13+的存储权限要求变更
- **解决方案**：适配Android 13+的存储权限要求，使用`Permission.photos`替代`Permission.storage`
