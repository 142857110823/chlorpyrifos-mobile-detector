# 光谱农药检测应用发布准备指南

## 1. 应用概述

光谱农药检测应用是一个基于Flutter开发的移动应用，能够通过分析PNG/JPG格式的光谱曲线图图片，识别四种不同农药（吡虫啉、扑虱灵、种衣剂、苄·二氣）的残留情况。系统采用MobileNetV2模型进行图像分类，实现离线、快速、准确的农药残留检测。

## 2. 构建应用安装包

### 2.1 构建Android应用

#### 2.1.1 配置构建环境
- 确保Flutter SDK已安装并配置正确
- 确保Android Studio已安装并配置正确
- 确保JDK 8或更高版本已安装

#### 2.1.2 构建命令
```bash
# 构建发布版本的APK
flutter build apk --release

# 构建拆分APK（针对不同架构）
flutter build apk --release --split-per-abi

# 构建Android App Bundle
flutter build appbundle
```

#### 2.1.3 构建结果
- **APK文件**：`build/app/outputs/flutter-apk/app-release.apk`
- **拆分APK**：`build/app/outputs/flutter-apk/app-armeabi-v7a-release.apk`、`build/app/outputs/flutter-apk/app-arm64-v8a-release.apk`
- **App Bundle**：`build/app/outputs/bundle/release/app-release.aab`

### 2.2 构建iOS应用

#### 2.2.1 配置构建环境
- 确保Flutter SDK已安装并配置正确
- 确保Xcode已安装并配置正确
- 确保MacOS环境已配置正确

#### 2.2.2 构建命令
```bash
# 构建发布版本的IPA
flutter build ios --release

# 构建模拟器版本
flutter build ios --debug
```

#### 2.2.3 构建结果
- **IPA文件**：需要通过Xcode导出
