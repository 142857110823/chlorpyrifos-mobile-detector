# 🚀 OpenClaw GUI - 新手入门指南

（3步就能用！）

---

## ✅ 第一步：安装依赖

在电脑上做这些：

1. 打开文件夹：`d:\王元元老师大创\OpenClaw_GUI_App\`
2. 在文件夹空白处按住 **Shift + 右键**
3. 选择 **"在此处打开 PowerShell 窗口"**
4. 输入这个命令，然后回车：

```powershell
npm install
```

等待几分钟，看到完成提示就好！

---

## ✅ 第二步：启动应用

在同一个 PowerShell 窗口里输入：

```powershell
npm start
```

嘭！✨ 应用窗口就弹出来了！

---

## ✅ 第三步：开始使用

应用打开后，您会看到4个页面：

### 📍 页面1：OpenClaw 配置
- 点 **"检查 OpenClaw"** 按钮
- 会显示您的 OpenClaw 版本
- 还能运行诊断、打开配置向导

### 📍 页面2：Discord Bot
- 输入您的 Bot Token 和 Client ID
- 点 **"保存配置"** 保存
- 以后不用再输了！

### 📍 页面3：工具
- 查看保存的配置
- 重置配置
- 打开项目文件夹

### 📍 页面4：关于
- 看看应用信息
- 访问官方文档

---

## 💡 小提示

### 如何打包成 .exe 安装程序？

如果想把这个应用发给朋友用：

```powershell
npm run build:win
```

安装程序会在 `dist/` 文件夹里！

### 应用关掉了怎么重新开？

1. 回到 `d:\王元元老师大创\OpenClaw_GUI_App\` 文件夹
2. 打开 PowerShell
3. 输入 `npm start`

---

## ❓ 遇到问题？

### npm install 特别慢？
- 网络问题，多等一会儿
- 或者检查网络连接

### npm start 报错？
- 先确认 `npm install` 成功完成了
- 看看错误提示是什么

### 按钮点了没反应？
- 确认 OpenClaw 已经安装
- 可以先在 PowerShell 试试 `openclaw --version`
