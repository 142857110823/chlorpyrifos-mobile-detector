## 💻 电脑端：Discord Bot 创建

### 第一步：访问 Discord 开发者门户

1. 打开电脑浏览器
2. 访问：**https://discord.com/developers/applications**
3. 使用你的 Discord 账号登录

### 第二步：创建新应用

1. 点击右上角的 **"New Application"** 按钮
2. 在弹出的框中输入 Bot 名字，比如：`我的超级Bot`
3. 点击 **"Create"** 按钮

### 第三步：创建 Bot 用户

1. 在左侧菜单点击 **"Bot"**
2. 点击 **"Add Bot"** 按钮
3. 在确认弹窗中点击 **"Yes, do it!"**

### 第四步：获取重要信息 ⭐

#### 获取 Bot Token（最重要！）
1. 在 Bot 页面中，找到 **"TOKEN"** 区域
2. 点击 **"Reset Token"** 按钮
3. 点击 **"Copy"** 按钮复制 Token
4. ⚠️ **立刻保存这个Token！** 关闭页面后就看不到了！

#### 获取 Client ID
1. 在左侧菜单点击 **"OAuth2"** → **"General"**
2. 复制 **"CLIENT ID"**，保存好

### 第五步：开启 Bot 权限

1. 回到 **"Bot"** 页面
2. 向下滚动找到 **"Privileged Gateway Intents"**
3. 打开以下三个开关：
   - ✅ **Presence Intent**
   - ✅ **Server Members Intent**
   - ✅ **Message Content Intent**
4. 点击 **"Save Changes"** 保存

### 第六步：邀请 Bot 到你的服务器

1. 点击左侧 **"OAuth2"** → **"URL Generator"**
2. 在 **"SCOPES"** 勾选：
   - ✅ `bot`
   - ✅ `applications.commands`
3. 在 **"BOT PERMISSIONS"** 勾选：
   - ✅ Send Messages
   - ✅ Read Message History
   - ✅ Read Messages/View Channels
   - ✅ Attach Files
   - ✅ Embed Links
   - ✅ Use Slash Commands
4. 复制页面底部的 **"GENERATED URL"**
5. 在浏览器打开这个链接
6. 选择你的服务器，点击 **"Continue"** → **"Authorize"**

## ⚙️ 电脑端：Bot 配置启动

### 第一步：配置环境变量

1. 打开文件夹：`d:\王元元老师大创\Discord_Bot_Project\config\`
2. 复制文件 `.env.example`，重命名为 `.env`
3. 右键用记事本打开 `.env` 文件
4. 填入刚才保存的信息：
   ```
   DISCORD_TOKEN=粘贴你的BotToken
   CLIENT_ID=粘贴你的应用程序ID
   ```
5. 保存文件

### 第二步：安装依赖

1. 在 `d:\王元元老师大创\Discord_Bot_Project\` 文件夹中按住 Shift + 右键
2. 选择 **"在此处打开 PowerShell 窗口"**
3. 输入以下命令并回车：
   ```powershell
   npm install
   ```
4. 等待安装完成（可能需要几分钟）

### 第三步：部署命令

1. 在同一个 PowerShell 窗口中输入：
   ```powershell
   node deploy-commands.js
   ```
2. 看到成功提示后继续

### 第四步：启动 Bot！

1. 在 PowerShell 中输入：
   ```powershell
   npm start
   ```
2. 看到 **"Bot已上线！"** 提示说明成功！

⚠️ **注意**：使用Bot时，这个PowerShell窗口不能关闭！

## 📱 手机端：Discord 操作指南

### 第一步：打开 Discord 手机应用

1. 在手机上打开 Discord 应用
2. 确保登录的是同一个账号

### 第二步：进入你的服务器

1. 点击左上角的三条横线菜单 ☰
2. 选择你邀请了 Bot 的服务器

### 第三步：使用斜杠命令

#### 方法一：在输入框使用

1. 进入任意频道
2. 点击底部的输入框
3. 输入 `/` 符号
4. 会弹出命令列表，选择你想要的命令
5. 点击发送

#### 方法二：直接输入命令

1. 在输入框直接输入完整命令，如：`/ping`
2. 点击发送

### 第四步：尝试三个示例命令

#### 命令1：/ping - 查看延迟

1. 输入 `/ping`
2. 发送
3. Bot 会回复显示网络延迟

#### 命令2：/hello - 打招呼

1. 输入 `/hello`
2. 发送
3. Bot 会友好地回复你

#### 命令3：/help - 获取帮助

1. 输入 `/help`
2. 发送
3. 会显示所有可用命令的说明

### 第五步：确认 Bot 在线

在服务器成员列表中，你应该能看到你的 Bot，名字旁边有绿色的在线标志 🟢。

## ❓ 常见问题解答

### Q1: Bot 没有反应怎么办？

**检查步骤：**
1. 电脑上的 PowerShell 窗口是否还开着？
2. Bot Token 是否正确填入？
3. Bot 是否已添加到服务器？
4. Bot 是否有频道的发言权限？

### Q2: 斜杠命令不显示？

**解决方法：**
1. 重新运行 `node deploy-commands.js`
2. 等待最多1小时让Discord同步（通常只需几分钟）
3. 尝试重启Discord应用

### Q3: 如何关闭 Bot？

**方法：**
- 在运行 Bot 的 PowerShell 窗口按 `Ctrl + C`

### Q4: Token 忘记了怎么办？

**解决方法：**
1. 回到 Discord 开发者门户
2. 进入 Bot 页面
3. 点击 **"Reset Token"** 重新生成
4. 更新 `.env` 文件中的 Token
5. 重启 Bot

### Q5: 可以在多个服务器使用吗？

**可以！**
- 使用同一个 OAuth2 邀请链接
- 将 Bot 添加到多个服务器即可
