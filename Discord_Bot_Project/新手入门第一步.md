# 🐣 超级简化版入门指南

（每步只做一件事，做完这步再看下步！）

---

## 🎯 今日目标：先在 Discord 上玩起来！

### 第一步：先看看你 Discord 上有没有服务器？

拿起手机，打开 Discord 应用：

1. 看左上角，有没有一个像"房子"或者"三个横线"的按钮？点一下
2. 看看有没有显示任何服务器？（服务器就像一个群聊组）

**如果你看到有服务器** → 太好了！直接跳到第三步

**如果你没有服务器** → 继续第二步

---

### 第二步：创建一个测试服务器（只用2分钟！）

在手机 Discord 上：

1. 点左上角的 "+" 号（添加服务器）
2. 选 "创建我自己的"
3. 选 "仅供我和我的朋友使用"
4. 服务器名字随便填，比如："我的Bot测试区"
5. 点 "创建"

✅ 好了！你有服务器了！

---

### 第三步：现在去电脑上，我们来创建 Bot！

（现在放下手机，坐到电脑前）

在电脑浏览器打开这个网址：
```
https://discord.com/developers/applications
```

用你的 Discord 账号登录。

---

### 第四步：创建一个应用（点6次鼠标！）

1. 点右上角的 **"New Application"** 按钮
2. 在弹出的框里，名字随便填，比如："我的第一个Bot"
3. 点 **"Create"**
4. 左边菜单点 **"Bot"**
5. 点 **"Add Bot"** 按钮
6. 点 **"Yes, do it!"**

✅ Bot 创建成功！

---

### 第五步：拿两个重要的号码（很重要！）

**第一个号码：Client ID**
1. 左边菜单点 **"OAuth2"** → **"General"**
2. 看到 "CLIENT ID" 了吗？点旁边的 "Copy" 复制
3. 打开电脑上的记事本，粘贴进去，写上："Client ID: xxxxx"

**第二个号码：Bot Token（最重要！）**
1. 左边菜单回到 **"Bot"**
2. 找到 "TOKEN" 那一块
3. 点 **"Reset Token"** 按钮
4. 点 **"Copy"** 复制
5. 赶快粘贴到记事本里，写上："Bot Token: xxxxx"
6. ⚠️ 这个号码千万别给别人！

✅ 两个号码都保存好了！记事本不要关！

---

### 第六步：开三个开关（很重要！）

还是在 **"Bot"** 页面：

1. 向下滚动，找到 **"Privileged Gateway Intents"**
2. 你会看到三个开关，**全部打开**：
   - ✅ Presence Intent
   - ✅ Server Members Intent
   - ✅ Message Content Intent
3. 点 **"Save Changes"** 保存

✅ 开关开好了！

---

### 第七步：把 Bot 邀请到你的服务器！

1. 左边菜单点 **"OAuth2"** → **"URL Generator"**
2. 在 **"SCOPES"** 区域，勾选两个：
   - ✅ `bot`
   - ✅ `applications.commands`
3. 向下滚动，在 **"BOT PERMISSIONS"** 区域，勾选这些：
   - ✅ Send Messages
   - ✅ Read Message History
   - ✅ Read Messages/View Channels
   - ✅ Attach Files
   - ✅ Embed Links
   - ✅ Use Slash Commands
4. 继续向下滚动，复制最底部的那个很长的链接（GENERATED URL）
5. 在浏览器新标签页打开这个链接
6. 选择你刚才创建的服务器（"我的Bot测试区"）
7. 点 **"Continue"** → **"Authorize"**

✅ Bot 已经加入你的服务器了！
