# 光谱图片农药识别系统预览方案

## 1. 应用概述

光谱图片农药识别系统是一个基于Flutter开发的移动应用，能够通过分析PNG/JPG格式的光谱曲线图图片，识别四种不同农药（吡虫啉、扑虱灵、种衣剂、苄·二氣）的残留情况。

## 2. 应用功能

### 2.1 主要功能
- **光谱图片检测**：支持拍摄或选择本地光谱图片进行农药识别
- **多农药识别**：能够识别四种不同的农药残留
- **实时反馈**：快速显示识别结果和置信度
- **用户友好界面**：简洁直观的操作流程

### 2.2 技术特点
- **离线识别**：所有处理在本地完成，无需网络连接
- **快速响应**：推理速度≤50ms，1秒内出结果
- **高精度**：基于MobileNetV2模型，识别准确率不断优化
- **轻量部署**：模型大小≤3MB，适合移动设备

## 3. 应用界面

### 3.1 启动页
- **设计**：全屏背景图片，半透明渐变遮罩
- **内容**：应用图标、应用名称、副标题、项目信息
- **功能**：3秒后自动跳转到主页

### 3.2 主页
- **设计**：底部导航栏，包含四个主要功能入口
- **功能**：
  - 首页：应用介绍和快捷操作
  - 检测：进入检测功能
  - 历史：查看检测历史记录
  - 设置：应用设置和配置

### 3.3 光谱图片检测页面
- **设计**：简洁的操作界面，包含图片预览、操作按钮和结果展示
- **功能**：
  - 拍摄光谱图：使用相机拍摄光谱曲线图片
  - 本地选择图片：从相册选择光谱曲线图片
  - 图片预览：显示选中的图片
  - 结果展示：显示识别结果和置信度
  - 图片输入规范提示：提供图片输入的标准要求

## 4. 操作流程

### 4.1 启动流程
1. 打开应用 → 显示启动页（3秒） → 自动跳转到主页

### 4.2 检测流程
1. 从主页进入检测页面
2. 选择“拍摄光谱图”或“本地选择图片”
3. 系统自动处理图片并进行识别
4. 显示识别结果和置信度
