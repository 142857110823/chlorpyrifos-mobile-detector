# 农药残留检测APP 用户指南

## 1. 应用介绍

农药残留检测APP是一款基于Flutter开发的移动应用，旨在帮助用户快速、准确地检测果蔬中的农药残留情况。应用结合了蓝牙技术和深度学习算法，提供了专业级的农药残留检测功能。

### 主要功能

- **实时检测**：通过蓝牙连接检测设备，实时采集光谱数据并分析农药残留
- **数据导入**：支持从CSV/Excel文件导入光谱数据进行分析
- **模拟检测**：内置模拟数据，无需设备即可体验检测功能
- **历史记录**：保存检测历史，支持查看和导出
- **AI可解释性**：提供AI分析结果的详细解释，增强结果可信度
- **PDF报告**：生成专业的检测报告，支持分享和打印

## 2. 快速开始

### 2.1 首次使用

1. **安装应用**：从应用商店下载并安装农药残留检测APP
2. **打开应用**：首次启动时，应用会引导您完成基本设置
3. **连接设备**：按照引导连接蓝牙检测设备（如果有）
4. **开始检测**：选择检测模式，输入样品信息，开始检测

### 2.2 基本操作流程

1. **进入检测页面**：点击首页的"开始检测"按钮
2. **选择检测模式**：根据实际情况选择"实时"、"导入"或"模拟"模式
3. **输入样品信息**：填写样品名称和类别
4. **开始检测**：点击"开始检测"按钮
5. **查看结果**：检测完成后，查看详细的检测结果
6. **操作结果**：可以选择生成PDF报告、分享结果或进行AI可解释性分析

## 3. 功能详解

### 3.1 首页

首页是应用的主界面，显示了以下信息：

- **设备状态**：显示蓝牙设备的连接状态
- **快捷操作**：提供"开始检测"、"历史记录"、"数据分析"三个快捷入口
- **检测统计**：显示今日检测次数、累计检测次数、安全样品数和风险样品数
- **最近检测**：显示最近5条检测记录

### 3.2 检测页面

检测页面是应用的核心功能页面，包含以下部分：

- **设备状态**：显示蓝牙设备的连接状态
- **样品信息**：输入样品名称和类别
- **数据来源**：选择检测数据的来源方式（实时、导入、模拟）
- **光谱数据**：显示采集到的光谱数据图表
- **检测进度**：显示检测的当前进度和状态
- **操作按钮**：根据检测状态显示不同的操作按钮
- **检测结果**：显示检测完成后的详细结果
- **结果操作**：提供AI可解释性分析、PDF报告生成、打印和分享功能

### 3.3 设备连接页面

设备连接页面用于搜索和连接蓝牙检测设备：

- **设备列表**：显示附近可发现的蓝牙设备
- **设备状态**：显示设备的连接状态
- **连接操作**：点击设备进行连接
- **连接历史**：显示最近连接过的设备

### 3.4 历史记录页面

历史记录页面用于查看和管理检测历史：

- **记录列表**：显示所有检测记录，按时间倒序排列
- **筛选功能**：支持按风险等级筛选记录
- **搜索功能**：支持按样品名称搜索记录
- **记录详情**：点击记录查看详细信息
- **批量操作**：支持批量删除和导出记录

### 3.5 设置页面

设置页面用于配置应用的各项参数：

- **通用设置**：语言、主题等基本设置
- **设备设置**：蓝牙设备相关设置
- **数据设置**：数据存储和导出相关设置
- **关于应用**：应用版本、版权信息等

## 4. 检测模式详解

### 4.1 实时模式

**适用场景**：有蓝牙检测设备的情况

**操作步骤**：
1. 确保蓝牙设备已开启并在范围内
2. 在设备连接页面连接设备
3. 选择"实时"模式
4. 输入样品信息
5. 点击"开始检测"
6. 等待设备采集数据并分析

### 4.2 导入模式

**适用场景**：已有光谱数据文件的情况

**操作步骤**：
1. 选择"导入"模式
2. 点击"选择文件"按钮
3. 选择CSV或Excel格式的光谱数据文件
4. 预览并确认数据
5. 输入样品信息
6. 点击"开始检测"
7. 等待分析完成

### 4.3 模拟模式

**适用场景**：无设备或测试应用功能的情况

**操作步骤**：
1. 选择"模拟"模式
2. 输入样品信息
3. 点击"开始检测"
4. 等待模拟分析完成

## 5. 结果解读

### 5.1 检测结果

检测结果页面显示了以下信息：

- **样品信息**：样品名称、检测时间、检测模式
- **风险等级**：安全、低风险、中等风险、高风险、严重超标
- **检测结果**：是否检测到农药残留
- **农药详情**：检测到的农药名称、浓度、最大残留限量
- **置信度**：AI分析结果的置信度

### 5.2 风险等级说明

| 风险等级 | 说明 | 建议 |
|---------|------|------|
| 安全 | 未检出农药残留或残留量在安全范围内 | 可以放心食用 |
| 低风险 | 检出少量农药残留，但在安全限量内 | 建议清洗后食用 |
| 中等风险 | 农药残留量接近或略超安全限量 | 建议充分清洗或浸泡后食用 |
| 高风险 | 农药残留量明显超标 | 建议谨慎食用，可考虑去皮或长时间浸泡处理 |
| 严重超标 | 农药残留严重超标 | 强烈建议不要食用，以免对健康造成危害 |

### 5.3 AI可解释性分析

AI可解释性分析页面提供了以下信息：

- **光谱波段贡献**：不同光谱波段对检测结果的贡献程度
- **关键波长**：对检测结果影响最大的波长
- **SHAP值分析**：AI模型预测的详细解释
- **特征重要性**：各特征对检测结果的重要性

## 6. 常见问题

### 6.1 设备连接问题

**问题**：无法连接蓝牙检测设备

**解决方案**：
1. 确保蓝牙设备已开启并电量充足
2. 确保设备在有效范围内（建议不超过10米）
3. 确保手机蓝牙已开启
4. 尝试重启设备和应用
5. 查看设备说明书，确认设备是否处于可发现模式

### 6.2 检测结果不准确

**问题**：检测结果与预期不符

**解决方案**：
1. 确保样品制备正确（如表面清洁、均匀放置）
2. 确保检测环境稳定（避免强光直射、振动等）
3. 尝试多次检测取平均值
4. 检查设备是否需要校准
5. 确保设备固件和应用版本为最新

### 6.3 应用崩溃或无响应

**问题**：应用崩溃或无响应

**解决方案**：
1. 尝试重启应用
2. 检查手机存储空间是否充足
3. 确保手机操作系统版本符合要求
4. 尝试清除应用缓存
5. 如问题持续，联系技术支持

### 6.4 数据导入失败

**问题**：无法导入光谱数据文件

**解决方案**：
1. 确保文件格式正确（CSV/Excel）
2. 确保文件内容符合要求（包含波长和强度数据）
3. 尝试简化文件名和路径
4. 检查文件大小是否超过限制

## 9. 术语解释

### 9.1 技术术语

- **光谱数据**：通过光谱仪采集的样品光谱信息，包含不同波长下的光强度
- **农药残留**：农药使用后在农产品中残留的微量农药原体、代谢物、降解物和杂质的总称
- **最大残留限量（MRL）**：法律允许的食品中农药残留的最高浓度
- **深度学习**：一种人工智能技术，用于从光谱数据中识别农药残留模式
- **AI可解释性**：解释AI模型如何做出预测的能力
- **SHAP值**：一种用于解释AI模型预测的方法

### 9.2 应用术语

- **实时模式**：通过蓝牙设备实时采集数据的检测模式
- **导入模式**：从文件导入数据的检测模式
- **模拟模式**：使用内置模拟数据的检测模式
- **风险等级**：根据农药残留量划分的风险程度
- **置信度**：AI分析结果的可信程度（0-1）
