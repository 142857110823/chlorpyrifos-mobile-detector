## Streamlit Web端改进详情

### 1. RandomForest模型替代CNN

**原方案**：使用numpy模拟的假CNN（numpy_cnn_classify, numpy_cnn_regress），输出随机数据。

**改进后**：使用scikit-learn的RandomForest，基于真实机器学习训练和推理。

```python
# requirements.txt
scikit-learn>=1.3.0

# 模型训练（@st.cache_resource 缓存）
@st.cache_resource
def _train_rf_models():
    X, y_cls, y_conc = _generate_training_spectra()
    clf = RandomForestClassifier(n_estimators=100, random_state=42, n_jobs=-1)
    clf.fit(X, y_cls)
    reg = RandomForestRegressor(n_estimators=100, random_state=42, n_jobs=-1)
    reg.fit(X, y_conc)
    return clf, reg
```

### 2. 合成训练数据生成

基于已有的PESTICIDE_PEAKS物理参数（各农药的特征吸收峰位置和宽度），内部生成训练数据：

- 每类农药约180个样本，共11类 + 无农药类
- 总计约2000条训练光谱
- 使用 `np.random.RandomState(42)` 确保确定性
- 光谱经预处理和特征提取后转为64维特征向量

### 3. 置信度改进

**原方案**：硬编码 `confidence = 0.7`

**改进后**：
- RandomForest模式：`confidence = float(np.max(clf.predict_proba(X)[0]))`（真实概率）
- 规则引擎模式：基于 match_ratios 和 intensity_quality 的数据驱动计算
- 混合模式：`w = conf^2` 自适应加权融合

### 4. SHAP解释改进

**原方案**：`np.random.uniform(-1, 1)` 随机生成

**改进后**：基于扰动的确定性SHAP
1. 将光谱分为16段
2. 逐段置零扰动
3. 用RF预测扰动后概率变化
4. 按局部梯度幅度分配重要性
5. 归一化到[-1, 1]范围
6. 完全确定性，无随机数

### 5. 特征重要性改进

**原方案**：`np.random.random()` 随机生成

**改进后**：直接使用 `rf_clf.feature_importances_`（基于Gini不纯度的真实特征重要性）

### 6. 置信区间改进

**原方案**：硬编码公式 `ci = (conf - 0.05, conf + 0.05)`

**改进后**：基于RF树方差的统计置信区间
```python
tree_preds = np.array([tree.predict_proba(X)[0] for tree in rf_clf.estimators_])
pred_std = float(np.std(np.max(tree_preds, axis=1)))
ci_lower = max(0, confidence - 1.96 * pred_std)
ci_upper = min(1, confidence + 1.96 * pred_std)
```

### 7. 移除的演示/模拟内容

- 移除 `numpy_cnn_classify()` 假CNN分类
- 移除 `numpy_cnn_regress()` 假CNN回归
- 移除 `generate_simulated_spectrum()` 模拟光谱生成
- 移除 "模拟演示" UI模式
- 移除所有 `time.sleep()` 假进度动画
- 移除所有硬编码估算公式
