## 🎯 今天只做一件事：运行配置向导！

### 第一步：打开PowerShell

在电脑上：
1. 按 `Win + X` 键
2. 选 "Windows PowerShell" 或 "终端"
3. 或者在开始菜单搜 "PowerShell" 打开

### 第二步：运行配置向导

在PowerShell里输入这个命令，然后回车：

```powershell
openclaw onboard
```

然后跟着提示一步步走就可以了！

## 💡 如果想先看看OpenClaw能做什么

在PowerShell里试试这些命令：

### 1. 查看版本
```powershell
openclaw --version
```
（您已经是 2026.3.2-beta.1 了！）

### 2. 查看所有命令
```powershell
openclaw --help
```

### 3. 运行健康检查
```powershell
openclaw doctor
```

### 4. 打开控制面板
```powershell
openclaw dashboard
```
