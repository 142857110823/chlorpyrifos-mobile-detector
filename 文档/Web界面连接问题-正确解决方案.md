2. 您现在看到的就是这个界面

**第三步：找到 Control UI 设置**
1. 看界面左侧菜单栏
2. 寻找"设置"、"配置"、"Control"、"Settings" 或类似的选项
3. 或者看界面右上角有没有设置按钮

**第四步：粘贴令牌**
1. 进入设置界面后
2. 找到"Gateway Token"、"令牌"、"API Key" 或类似的输入框

4. 点击"保存"、"连接"或"确认"

**第五步：验证连接**

### 方案二：使用 Tokenized Dashboard URL

如果方案一不行，试试这个：

#### 🔍 如何获取 Tokenized URL

**第一步：检查 OpenClaw 启动日志**
1. 看运行 OpenClaw 的终端窗口
2. 向上滚动，寻找启动时的输出
3. 寻找类似这样的 URL：
   ```
   Tokenized dashboard URL: http://127.0.0.1:18789/?token=xxxxx
   ```
   或者
   ```
   Dashboard URL with token: http://127.0.0.1:18789/#token=xxxxx
   ```

**第二步：直接访问带令牌的 URL**
1. 如果找到了这样的 URL
2. 直接在浏览器中打开那个完整的 URL
3. 应该就能自动连接了！
